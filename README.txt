Air Species Properties
Author: Gary Hammock, PE  (2014)

================================================================================
                                DESCRIPTION
================================================================================

This Windows(R) application can calculate the thermodynamic and transport
properties of the gas species found in equilibrium air using the fundamentals of
quantum thermodynamics.

The program accepts inputs and generate outputs in standard SI units.

================================================================================
                                   USAGE
================================================================================

1.) Double click the executable file to launch the application.
2.) Enter the temperature and (partial) pressure of the state the species is in.
3.) Select which of the available gas species you are interested in.
4.) Click the "Calculate" button.
5.) When finished, click the "Exit" button (or the standard Windows close [X]
    button to exit the application.

5a.) The "About" option displays the licensing content that is also found in
     the accompanying "license.txt" file.  This window also contains a button
     labeled "References" that displays the references used in the calculation
     of the properties (the same references as those found below).
5b.) The "File" menu allows for saving the output data as a plain text file
     (.txt).
